Same Element - Sibling Relationship

The Character Energies have the same element. There is a potential for a deep emotional connection as well as a very good mutual understanding on an emotional level. This can help to create more harmony in the relationship, especially if the fundamental Energies are antagonistic.

In romantic relationships, sharing the same Character Energy element can lead to less chemistry or spark. However, if the other Energies are in a Supportive Transformation or Challenging Transformation then the chemistry may very well be more noticeable.
